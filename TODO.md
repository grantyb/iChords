Sharing songs:
    I'd like a way to synchronise a song between two devices running the app. Please make a plan for this feature:
     Add a "share" icon in the header of the play view. When tapped, show nearby instances of the app and allow the user to select one of those other instances to send the song data to. The other app will "hear" the share request and pop up an "accept song" prompt that allows the user to accept or ignore the data. The prompt with show the name and artist of the song and the album art. Accepting the song will cause a new entry to be added to the library on the receiving device. The data will be serialised from the "source" app, and will include the song data and song beat data.
